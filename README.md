# tanglegram_tree

Personal research scripts for visualizing paired influenza gene trees as a
tanglegram.

## Contents

- `scripts/plot_ha_pb2_tanglegram.py`: draw an HA/PB2 tanglegram from two
  Newick trees and a metadata table.

## Notes

This repository is mainly for my own academic research workflow. The scripts
are shared for record keeping and non-commercial scientific reference.

Please contact the author before reusing the code in other projects,
redistributing it, or using it for commercial purposes.
